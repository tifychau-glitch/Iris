Your step-by-step guide to getting started with Iris.

Welcome to Iris. You've just signed up for an accountability coach who doesn't forget anything, doesn't let you off easy, and genuinely wants you to win. She's sharp, she's warm, and she's yours.

This email is your complete setup guide. Everything you need is here. Follow the steps in order and you'll be up and running in about 15 minutes.

Before you begin

What you'll need

Make sure you have these four things before you start:


A Mac. Iris's automatic scheduling runs on macOS. Windows support is not included in this version.


Telegram. Free app available at telegram.org or in the App Store. If you already have it, you're set. If not, download and create a free account now.


An Anthropic API key. This is what powers Iris's intelligence. If you already use Claude, you have one. If not, get one at console.anthropic.com. It's pay-as-you-go — Iris's daily usage costs pennies.


Terminal. Already on your Mac. You'll find it in Applications > Utilities, or search "Terminal" in Spotlight. No coding experience needed — you're just copy-pasting commands.

1 Download and Install Iris ~3 min

Download your files

Use the download link from your purchase confirmation. You'll get a zip file called iris-product.zip or similar.

Save it somewhere permanent — like your Desktop or Documents folder. Not Downloads. Files in Downloads tend to get cleared, and Iris's databases live inside this folder.

Unzip the folder

Double-click the zip file to extract it. You'll have a folder called iris-product (or similar). Move it to where you want to keep it permanently.

Open Terminal and navigate to the Iris folder

Open Terminal. Then type cd followed by a space, then drag your Iris folder directly into the Terminal window. It will auto-fill the full path. Press Enter.

cd /Users/yourname/Desktop/iris-product
Dragging the folder into Terminal is the easiest way to get the exact path without typing it.

Run the installer

Copy and paste this command, then press Enter. It installs everything Iris needs. Takes about 2 minutes.

bash setup.sh

You'll see packages downloading. When it finishes and you see a prompt again, you're done. If it asks you for anything, just press Enter to accept the defaults.

2 Create Iris's Telegram Bot ~3 min

Iris communicates through her own private Telegram bot. You'll create it now using Telegram's official bot builder.

Find @BotFather on Telegram

Open Telegram. In the search bar, type @BotFather. It should be the first result, with a blue checkmark. Open that chat.

Create a new bot

Send this message to BotFather:

/newbot

BotFather will ask you two questions.

Answer BotFather's questions

First question — "What's the name of your bot?"

Type: Iris


Second question — "Choose a username for your bot."

Pick something unique that ends in _bot. Examples:

myiris_bot
irisformike_bot
iriscoach_bot
Bot usernames are globally unique across all of Telegram. If it says the name is taken, try something slightly different.

Copy your bot token

After you choose a username, BotFather will send you a message containing your bot token. It looks like this:

7834521097:AAFqrTzSeLIoqMnXnpP8WxQDtUv_yZkRCmE

Copy the entire token and save it somewhere temporarily. You'll need it in the next step.

Keep this token private. Anyone who has it can control your bot.
3 Add Your Credentials ~2 min

Find the .env file

Inside your Iris folder, there's a file called .env. It starts with a dot, which means it's hidden by default.


To see hidden files in Finder, open the Iris folder and press:

Command + Shift + .

Hidden files will appear. Look for .env.

Open and fill in the .env file

Right-click .env and open it with TextEdit or any text editor. You'll see:

TELEGRAM_BOT_TOKEN=# paste your token here
ANTHROPIC_API_KEY=# paste your Anthropic key here
TELEGRAM_CHAT_ID=# leave this blank

Fill in your Telegram bot token and your Anthropic API key. Leave TELEGRAM_CHAT_ID blank — Iris fills it in automatically after you first message her.

Make sure there are no spaces before or after the values. Just: KEY=value with nothing extra.

Save the file

Save and close the .env file. You're done with credentials.

4 Start Iris ~1 min

Activate the environment and start the bot

Back in Terminal (still in your Iris folder), run these two commands. Copy them exactly.

source .venv/bin/activate
python3 .claude/skills/iris-telegram/scripts/iris_bot.py

If everything worked, you'll see:

[iris_bot] Iris is listening on Telegram...
Always run source .venv/bin/activate first before any Iris commands. This activates the virtual environment Iris needs to run.

Keep this Terminal window open

The bot runs as long as this Terminal window is open. Don't close it. If you do, Iris will stop responding until you start her again. Later in this guide (Part 6), you'll set her up to run automatically in the background — you won't need to do this manually every time.

5 Say Hello — Onboarding with Iris ~5 min

Find your bot on Telegram

Open Telegram. In the search bar, type the bot username you created (e.g. @myiris_bot). Open the chat.

Send your first message

This is the most important step that people miss: you have to type something first. Iris is listening, but she won't speak until you initiate. It doesn't matter what you say.

hi

The moment you send that message, two things happen: Iris introduces herself, and your chat ID gets saved so check-ins work going forward.

If Iris doesn't respond within 30 seconds, go back to Terminal and make sure iris_bot.py is still running (you should see the "Iris is listening" message).

Answer her onboarding questions

Iris will walk you through 6 quick questions. Here's what she's asking and what she's doing with your answers:


  • 👤
    What should I call you?

    Your name. She'll remember it and use it. No need to be formal.

  • 🎚️
    Your max intensity level (1-5)

    This is the ceiling of how harsh she'll ever get with you. 1 is warm and encouraging. 5 is full drill sergeant. She starts gentle with everyone and only escalates based on your actual behavior. Pick the highest you're honestly willing to tolerate when things get rough.

  • ⏰
    Your wake and sleep times

    Just rough times, like "7am and 11pm." She uses this to schedule check-ins at reasonable hours and figure out what counts as "today."

  • 📬
    Your check-in times

    She'll suggest a morning, midday, and evening time based on your schedule. You can say yes or give her custom times. These are when she'll message you proactively — you don't have to remember to check in.

  • 🤬
    Swearing preference

    At the higher accountability levels, Iris gets more colorful. She asks if that's okay. Honest answer recommended — this isn't about being polite, it's about what actually works for you.

  • 🎯
    Your first commitment

    One thing you're going to do today. She logs it, tracks it, and will ask you about it. Make it real — this is where it starts.

If you get interrupted mid-onboarding — app closes, phone dies, anything — just come back and say hi again. Iris picks up exactly where you left off.
Quick tip: IRIS works best when you're specific. Tell her what's actually going on — not the polished version. The more real you are, the more useful she gets.

You'll know you're set up when she says:

"Logged. I'll remember that. [Name], I'm watching now. Welcome to Iris."

"People treat accountability like it's supposed to feel comfortable. It's not. If it felt comfortable, you wouldn't need me. I'm here for the gap between what you say you'll do and what you actually do. That's my lane. Let's see what we're working with."

— Iris
6 Set Up Automatic Check-Ins ~3 min — recommended

This is what makes Iris proactive. Without this step, you have to reach out to her. With it, she reaches out to you — morning, midday, and evening — automatically.

Find your Iris folder path

In Terminal, type this command to print the full path to your Iris folder:

pwd

Copy that output. It will look something like: /Users/yourname/Desktop/iris-product

You'll need to replace REPLACE_WITH_FULL_PATH in the commands below with this value.

Create the morning check-in job

Run this in Terminal (replacing the path with your actual path):

cat > ~/Library/LaunchAgents/com.iris.checkin.morning.plist << 'EOF'
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
  <key>Label</key>
  <string>com.iris.checkin.morning</string>
  <key>ProgramArguments</key>
  <array>
    <string>REPLACE_WITH_FULL_PATH/.venv/bin/python3</string>
    <string>REPLACE_WITH_FULL_PATH/.claude/skills/iris-telegram/scripts/iris_checkin.py</string>
    <string>--type</string>
    <string>morning</string>
  </array>
  <key>StartCalendarInterval</key>
  <dict>
    <key>Hour</key>
    <integer>8</integer>
    <key>Minute</key>
    <integer>0</integer>
  </dict>
</dict>
</plist>
EOF

Then load it:

launchctl load ~/Library/LaunchAgents/com.iris.checkin.morning.plist
Add an evening check-in by repeating the same steps with a file named com.iris.checkin.evening.plist, changing the Hour to 20 and the --type to evening.

Test it (optional but recommended)

Run this to send a real check-in message right now:

source .venv/bin/activate
python3 .claude/skills/iris-telegram/scripts/iris_checkin.py --type morning

You should get a Telegram message from Iris within a few seconds.

7 Run Iris in the Background ~2 min — recommended

Right now, Iris only responds when that Terminal window is open. This step makes her run in the background automatically — she starts when your Mac starts and stays running without you doing anything.

Create the background service

Run this in Terminal (replacing the path):

cat > ~/Library/LaunchAgents/com.iris.bot.plist << 'EOF'
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
  <key>Label</key>
  <string>com.iris.bot</string>
  <key>ProgramArguments</key>
  <array>
    <string>REPLACE_WITH_FULL_PATH/.venv/bin/python3</string>
    <string>REPLACE_WITH_FULL_PATH/.claude/skills/iris-telegram/scripts/iris_bot.py</string>
  </array>
  <key>KeepAlive</key>
  <true/>
  <key>RunAtLoad</key>
  <true/>
</dict>
</plist>
EOF

launchctl load ~/Library/LaunchAgents/com.iris.bot.plist

After this, you can close the Terminal window with iris_bot.py running. Iris will keep responding in the background.

Daily Command Reference

These are the commands you'll use every day. You can also just talk to Iris in plain English and she'll respond in character.

You type What happens
done 1 Marks commitment #1 as complete. Use the number from your list.
done gym Marks a commitment by keyword instead of number.

help Shows the full command reference inside Telegram.
anything else Iris responds in character. You can have a real conversation with her.

Common Questions

Iris isn't responding to my messages.

Make sure iris_bot.py is running (check your Terminal or confirm the background service is loaded). Also verify your bot token is correctly entered in .env with no extra spaces.

I get a "ModuleNotFoundError" or missing package error.

You're running outside the virtual environment. Run source .venv/bin/activate first, then try again. You need to do this every time you open a new Terminal session and run Iris manually.

I see "TELEGRAM_CHAT_ID not set."

Open Telegram, find your bot, and send it any message. Iris saves your chat ID automatically after your first message. This is why you must message the bot before check-ins will work.

Check-ins aren't coming through.

Three possible causes: (1) You haven't sent Iris a first message yet, so your chat ID isn't saved. (2) The launchd plist has the wrong path — make sure it points to .venv/bin/python3 inside your Iris folder, not /usr/bin/python3. (3) Test it manually with python3 .claude/skills/iris-telegram/scripts/iris_checkin.py --type morning.

I closed Terminal and now Iris won't respond.

If you haven't set up the background service yet (Part 7), you'll need to restart the bot: source .venv/bin/activate then python3 .claude/skills/iris-telegram/scripts/iris_bot.py. Set up the background service to avoid this.

The bot username I wanted is already taken.

Bot usernames are globally unique on Telegram. Add something personal to make it unique — your name, initials, or a random word. As long as it ends in _bot, it works.

Can I use Iris on Windows?

Not with this version. The automatic check-in scheduling uses macOS's launchd. You can still run the bot manually on Windows, but the scheduled check-ins won't work without modification.

How much does the Anthropic API cost?

Very little. Iris uses Claude Haiku for quick replies, which is one of the cheapest models available. Typical daily usage costs a few cents. You can monitor your usage at console.anthropic.com.

I want to reset everything and start fresh.

Delete the files in the data/ folder inside your Iris directory. That clears all commitments, memory, and calibration. The next time you message the bot, Iris will treat you as a new user and restart onboarding.

What if I want to add more skills later?

Iris is built to grow with you. She ships with 18 skills, but you can add new ones anytime. If you need Iris to handle something she doesn't do yet — a specific workflow, a new tool, a recurring process — just tell her: "Create a skill for [describe what you need]." She'll build it, add it to her skill set, and it works alongside everything she already knows. Your version of Iris becomes more capable over time, tailored to exactly how you work.
