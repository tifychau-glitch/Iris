# Post-Purchase Email — Getting Started with IRIS

**Subject line:** You're in. Here's how to set up IRIS (10 minutes).

---

Hey!

You just got IRIS. Let's get you set up — it takes about 10 minutes.

There are 5 steps. Do them in order.

---

## Step 1: Install the Claude app

IRIS runs inside an app called Claude. Download it here:

**👉 [Download Claude](https://claude.ai/download)**

Install it like any other app. Open it and sign in with your Claude account.

*(If you don't have a Claude account yet, create one at [claude.ai](https://claude.ai). You'll need a Pro subscription — $20/month.)*

---

## Step 2: Download and unzip IRIS

Download the IRIS zip file from your purchase confirmation email. Then:

- **Mac:** Double-click the zip file. It'll create a folder called "Iris-Pro."
- **Windows:** Right-click the zip file → "Extract All."

Put the folder somewhere easy to find — your Desktop or Documents folder works.

**Mac users:** The first time you open files from the folder, your Mac might say it can't be opened because it's from an unidentified developer. If that happens:
1. Go to **System Settings → Privacy & Security**
2. Scroll down — you'll see a message about the blocked file
3. Click **"Open Anyway"**

---

## Step 3: Check for requirements

IRIS needs a few things installed on your computer to work. The setup script will try to install them automatically, but some may need you to download them manually.

**What IRIS needs (called "dependencies" — just programs IRIS relies on):**

- **Git** — a tool that Claude Code uses behind the scenes. Most Macs already have it. Windows usually doesn't — [download Git here](https://git-scm.com) if needed.
- **Python** — a programming language IRIS uses to run. Most Macs already have it. Windows users: [download Python here](https://python.org). **Important: during install, check the box that says "Add to PATH."**

You might already have both. If you're not sure, don't worry — when you run IRIS in the next step, it'll tell you if anything is missing and help you get it.

---

## Step 4: Start IRIS

Open the **Iris-Pro** folder and find the **"Start Here"** folder inside it. Then:

- **Mac:** Double-click **"Start IRIS.command"**
- **Windows:** Double-click **"Start IRIS.bat"**

Here's what happens:
1. It checks that everything is installed (and helps you install anything that's missing)
2. The IRIS dashboard opens in your web browser
3. The Claude app opens

**If it asks you to install something:** follow the instructions on screen, install it, then double-click "Start IRIS" again. It picks up where it left off.

**First time only:** the dashboard will ask you to create a password. Pick something you'll remember — you'll use it to access your settings.

---

## Step 5: Connect IRIS to your folder

Once Claude opens:

1. Click the **"Code"** tab at the top of the Claude app
2. Set the environment to **"Local"**
3. Click **"Select folder"** and choose the **Iris-Pro** folder you unzipped in Step 2
4. Type **"hi"** in the message box and press Enter

**That's it.** IRIS will introduce herself and walk you through the rest — connecting Telegram, getting to know your business, and setting everything up. Just follow her lead.

---

## What to expect

The first conversation takes about 10 minutes. IRIS will:

- Ask about you and what you're building
- Walk you through connecting Telegram (so she can check in with you outside the app)
- Set up your dashboard with your integrations
- Create your personal vault (your second brain — you'll be able to view it in Obsidian later)

After that, you're live. IRIS will check in with you, track your commitments, and help you close the gap between what you say you'll do and what you actually do.

---

## If something goes wrong

- **"Git is not installed" / "Python is not installed"** — Download from the link IRIS shows you, install it, then double-click "Start IRIS" again.
- **Python installed but IRIS says it's not** — You probably missed the "Add to PATH" checkbox during Python install. Uninstall Python, reinstall it, and check that box this time.
- **Dashboard won't open** — Try going to http://localhost:5050 in your browser manually.
- **Claude app opens but IRIS doesn't respond** — Make sure you clicked the "Code" tab, set environment to "Local", and selected the Iris-Pro folder.
- **Stuck?** Reply to this email and I'll help you personally.
