## Folder purposes

- **`Efforts/`** — Active projects. One file per project. Close them out when finished (move to `Atlas/archive/` or just leave them).
- **`Atlas/`** — Reference material you're NOT actively working on. Saved articles, research, competitor notes, book highlights.
- **`Calendar/`** — Daily notes. `YYYY-MM-DD.md` format.
- **`Concepts/`** — Synthesized insights. In Phase 3, IRIS proposes these based on patterns she notices.
- **`maps/`** — Meta files about the vault itself. This file and `iris-rules.md`.
